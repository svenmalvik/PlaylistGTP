Playlist from YouTube copyright claims

How to Use

Click the "AI magic" button to generate a playlist. Once the response is displayed, you can copy it to the clipboard using the "Copy to Clipboard" button that will appear after a valid output is generated.

Note: This extension only works within YouTube Studio using the following URL format: studio.youtube.com/video/<VIDEO_ID>/copyright

API endpoint and API key can be set in the extension's configuration.
